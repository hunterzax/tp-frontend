## Getting Started

### running on dev mode
`npm run dev`

### running on prod
`npm run start`

Open `http://localhost:5001` with your browser to see the result.

## Command to build in local (Macbook)

`1. docker build -t nx:dev_tpa_frontend_v1.0.55 --platform linux/amd64 .`
### then
`2. docker tag nx:dev_tpa_frontend_v1.0.53 kompkw/onlybuild:dev_tpa_frontend_1053`
### then
`3. docker push kompkw/onlybuild:dev_tpa_frontend_1053`
